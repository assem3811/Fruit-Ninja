Fruit Ninja Project By Java.

How to play?
1.	Open the jar file of the game
2.	Click Start
3.	Select the difficulty of the game
4.	Keep slicing the dropping fruits and be aware of the bombs, also try to not let a fruit go without slicing it.
5.	Beat the High sore
6.	Enjoy!
